Reference
=========

This reference manual details functions, modules, and objects included in dbdicom, describing what they are and what they do.

Many docstrings contain example code, which demonstrates basic usage of the routine. The examples assume that dbdicom and numpy are imported with::

  >>> import numpy as np
  >>> import dbdicom as db
